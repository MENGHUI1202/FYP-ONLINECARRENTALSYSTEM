# FYP Online Car Rental System

KH Car Rental is a PHP/MySQL car rental system for the FYP project.

## Main Folders

- `NEW_CAR_RENTAL_SYSTEM/` - customer website
- `admin/` - admin dashboard
- `database/` - SQL backup files
- `assets/` - shared CSS and car images
- `includes/` - shared admin/includes files

## Database

Use this database name:

```text
car_rental_system_unified_copy_20260609_001225
```

The latest SQL backup is:

```text
database/car_rental_system_unified_copy_20260609_001225_with_database.sql
```

## Local Setup

1. Put this folder inside XAMPP `htdocs`.
2. Start Apache and MySQL in XAMPP.
3. Import the SQL file into phpMyAdmin or MySQL.
4. Open:

```text
http://localhost/onlinecarrentalsystem/NEW_CAR_RENTAL_SYSTEM/homepage.php
```

## Note

Customer-uploaded KYC documents and profile upload files are ignored from Git for privacy.
